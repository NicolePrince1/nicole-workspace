# Oviond Trial Onboarding — Proposed New Email Sequence

**Author:** Nicole Prince, VP Marketing

**Automation:** 1.1 Trialling Simplified (15 Days)
**Trigger:** Subscriber joins group "1.1. 🚀 Trial Users - All (0 to 15 Days)"

---

## Design Principles

1. **One email, one action** — every email has exactly one CTA button
2. **Logical order** — connect data → build report → customize → share → convert
3. **Spaced properly** — 9 emails over 15 days (not 7 in 6 days)
4. **Outcome-focused subjects** — sell the result, not the feature
5. **Consistent voice** — all from Chris (founder, personal, warm)
6. **Social proof woven in** — real outcomes, not just features

---

## Technical Notes for Implementation

### Conditions to add:
- **Before Email 2 (Day 1):** Check if user has already connected a data source → if yes, skip to Email 3
- **Before Email 3 (Day 3):** Check if user has created a report → if yes, skip to Email 5
- **Before Email 8 (Day 13):** Check subscription_status = "active" → if yes, route to 3.1 Paying Customer Onboarding

### End of sequence routing:
- Check subscription_status = "active" → Move to group 3.1 (Paying Customer Onboarding)
- Check subscription_status = "trial_expired" → Move to group 2.1 (Trial Expired Re-Engagement)
- Else → Move to group 4.1 (Lapsed Customer Reactivation)

### Sender settings:
- All emails from: Chris from Oviond <hi@getoviond.com>
- Reply-to: hi@getoviond.com
- Language: English (ID 4)
- Track opens: Yes
- Google Analytics tracking: Yes (add UTM: source=sequenzy, medium=email, campaign=trial-onboarding-v2)

### Merge fields used:
- `{$name}` — subscriber first name
- `{$company}` — subscriber company name (from Stripe sync)

### A/B Testing recommendation:
Don't replace the current sequence entirely. Run the new sequence for 50% of new trials for 30 days, then compare:
- Open rate
- Click rate
- Trial-to-paid conversion rate
- Unsubscribe rate
