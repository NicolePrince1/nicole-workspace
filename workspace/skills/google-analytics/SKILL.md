## Available Dimensions (Common)

| Dimension | Description |
|-----------|-------------|
| `date` | YYYYMMDD format |
| `sessionDefaultChannelGroup` | Organic Search, Paid Search, Direct, etc. |
| `sessionSourceMedium` | google / organic, google / cpc, etc. |
| `sessionSource` | Source only |
| `sessionMedium` | Medium only |
| `sessionCampaignName` | UTM campaign name |
| `country` | User country |
| `city` | User city |
| `deviceCategory` | desktop, mobile, tablet |
| `operatingSystem` | OS name |
| `browser` | Browser name |
| `landingPagePlusQueryString` | Entry page path |
| `pagePath` | Page path |
| `pageTitle` | Page title |
| `newVsReturning` | new or returning |
| `eventName` | Event name |

## Available Metrics (Common)

| Metric | Description |
|--------|-------------|
| `sessions` | Total sessions |
| `totalUsers` | Total users |
| `newUsers` | New users |
| `activeUsers` | Active users |
| `screenPageViews` | Page views |
| `bounceRate` | Bounce rate (0-1) |
| `averageSessionDuration` | Avg session length (seconds) |
| `engagementRate` | Engagement rate (0-1) |
| `conversions` | Total conversions |
| `eventCount` | Total events |
| `eventValue` | Total event value |

## Date Ranges

- Relative: `today`, `yesterday`, `7daysAgo`, `14daysAgo`, `28daysAgo`, `30daysAgo`, `60daysAgo`, `90daysAgo`
- Absolute: `YYYY-MM-DD`
- Comparison: pass two dateRanges with `name` fields

## Notes

- GA4 Data API: `analyticsdata.googleapis.com/v1beta`
- GA4 Admin API: `analyticsadmin.googleapis.com/v1beta`
- Rate limit: 60 requests per minute per project
- Property ID required for all data queries
- Date format in responses: YYYYMMDD
- bounceRate and engagementRate are 0-1 (multiply by 100 for percentage)
- averageSessionDuration is in seconds
