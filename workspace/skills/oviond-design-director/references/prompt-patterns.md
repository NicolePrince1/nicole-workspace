# Prompt Patterns

Use these patterns to create consistent generation prompts. Replace bracketed fields with the task-specific brief.

## 1. Brand-led concept prompt

Use when the asset needs strong Oviond identity but flexible execution.

Template:

```text
Create a [asset type] for Oviond, a clean and modern SaaS reporting platform for marketing agencies. Objective: [goal]. Audience: [audience]. Core message: [message]. Show this with a visual style that feels clear, commercially credible, uncluttered, and premium without being flashy. Composition: [composition]. Keep strong hierarchy, generous spacing, and one dominant focal point. Palette direction: [palette]. If text appears, keep it minimal and bold. Avoid generic stock-photo aesthetics, noisy backgrounds, cluttered dashboards, rainbow gradients, cheesy startup illustrations, and overdesigned decorative elements.
```

## 2. Conversion ad prompt

Use for paid social and direct-response creative.

Template:

```text
Design a high-clarity paid social ad for Oviond. Audience: [audience]. Pain point: [pain]. Promise/value proposition: [value prop]. Desired action: [CTA]. Build for fast mobile comprehension with bold contrast, simple composition, one dominant message, and a clean CTA zone. Visual direction: [style]. Include visual cues that imply reporting clarity, agency efficiency, and reduced manual work. Avoid clutter, tiny UI details, weak focal points, excessive text, and soft low-contrast layouts.
```

## 3. Web hero visual prompt

Use for landing pages, homepage sections, and product marketing pages.

Template:

```text
Create a website hero visual for Oviond, a reporting platform for marketing agencies. Support this headline/theme: [headline/theme]. The image should communicate [idea/outcome] while leaving room for surrounding web copy and CTA buttons. Style: polished SaaS, modern, clean, trustworthy, commercially sharp. Composition: [composition]. Keep the scene readable, spacious, and easy to integrate into a landing page. Avoid busy compositions, fake-complex enterprise dashboards, and abstract art that says nothing.
```

## 4. Stat card / infographic prompt

Use for data-led content moments.

Template:

```text
Create a clean stat-card style graphic for Oviond centered on this key insight: [stat/insight]. Audience: [audience]. Make the number or key claim the unmistakable focal point. Use a modern, minimal SaaS design language with strong hierarchy, disciplined spacing, and restrained color. Keep supporting copy minimal. Avoid clutter, over-illustration, tiny labels, and multi-message confusion.
```

## 5. Event / webinar prompt

Use for registrations and promo graphics.

Template:

```text
Design a webinar or event promo graphic for Oviond. Event theme: [theme]. Audience: [audience]. Prioritize a clear headline area, clean event-detail zones, and a polished SaaS aesthetic that feels professional and useful. Composition should support title, date/time, speaker names, and CTA placement without overcrowding. Avoid poster-style chaos, too many decorative elements, and low-legibility typography.
```

## Variation angles

When generating variants, change only 1-2 major variables at once. Good variation axes:
- lighter vs darker base
- product-led vs conceptual visual
- more structured grid vs more open composition
- sharper contrast vs softer premium tone
- minimal text vs text-free visual

## Negative constraints bank

Mix in relevant exclusions:
- no generic stock-photo office scenes
- no noisy gradient overload
- no tiny illegible dashboard widgets
- no excessive floating UI cards
- no cartoonish mascots
- no dribbble-style fluff without message clarity
- no fake 3D chrome objects unless explicitly requested
- no cluttered multi-message layouts

## Prompt packaging standard

Before generation, produce:
- **Master prompt**
- **Negative constraints**
- **Variation A/B/C notes**
- **Placement reminder**
