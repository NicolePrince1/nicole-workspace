# Prompt Recipes

Use these recipes as starting structures. Replace bracketed fields with the actual brief.

## 1. Overlay-safe paid social concept

```text
Create a [format] paid social concept for [brand]. Objective: [goal]. Audience: [audience]. Core message: [message]. Show one dominant focal point with a clean, high-contrast composition designed for fast mobile comprehension. Leave a clear, uncluttered zone for headline and CTA overlay in the [top/bottom/left/right] area. Style: [style]. Palette: [palette]. Avoid stock-photo clichés, clutter, tiny UI details, weak focal points, messy backgrounds, and generic AI gloss.
```

## 2. Short-copy in-image ad

```text
Design a [format] marketing image for [brand]. Audience: [audience]. The image must clearly communicate: "[exact short line]". Render that exact phrase cleanly and legibly as the primary headline. Composition: [composition]. Visual style: [style]. Keep the rest of the layout simple so the headline remains dominant. Add a clean secondary CTA area only if needed. Avoid extra copy, decorative clutter, warped letters, inconsistent typography, and low contrast.
```

## 3. Blog cover or content graphic

```text
Create a polished blog cover image for [brand]. Article theme: [theme]. Audience: [audience]. Build a clean, modern composition that communicates [idea]. Leave safe space for a headline overlay. Style: [style]. Keep the visual commercially credible, spacious, and useful rather than noisy or overly abstract. Avoid sensational thumbnail energy, busy collage layouts, cluttered UI fragments, and generic stock-business scenes.
```

## 4. Edit an existing asset

```text
Edit the provided image. Keep [elements to preserve]. Change [elements to change]. New goal: [goal]. Preserve the overall composition unless required otherwise. Improve [clarity/contrast/premium feel/brand fit/text zone/etc.]. Keep the result clean, commercially credible, and ready for [placement]. Avoid introducing new objects, random decorations, spelling errors, or unnecessary layout changes.
```

## 5. Multi-turn refinement note

Use short follow-ups like:
- keep the same composition, make it more premium
- preserve the focal object, simplify the background
- keep everything but increase whitespace for headline overlay
- keep the layout, change the palette to [palette]
- keep the same concept, make the CTA area more obvious

## Negative constraint bank

Mix in what matters:
- no stock-photo office scenes
- no meeting-room clichés
- no cluttered floating dashboard cards
- no rainbow gradients
- no decorative nonsense
- no tiny unreadable text
- no weak contrast
- no generic template look
- no extra objects beyond the brief
