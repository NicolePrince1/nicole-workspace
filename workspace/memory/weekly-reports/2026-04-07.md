# Oviond Weekly Summary | 2026-04-07

## Headline

We made real progress last week, but the business is still in a messy middle: paid acquisition is more operational than it was, retention is now correctly exposed as the bigger problem, and measurement is still noisy enough to lie if we let it.

## Scorecard vs goals

- **Revenue / MRR trajectory:** **mixed**
  - Working MRR truth is still roughly **$28.7k**, with the long-range goal still **$100k by Dec 2026**.
  - Rolling 7-day paid invoice revenue is **$5,679.59**, down **18.4%** vs the prior 7-day window (**$6,958.55**).

- **Retention / churn:** **off track**
  - This is now the core strategic problem.
  - Churn appears to be driven more by **unused / switched / complexity / client-confidence issues** than by failed payments alone.
  - There are **48 active trials**, up slightly, but **10 expire in the next 3 days**, which makes activation and rescue work urgent.

- **US acquisition engine:** **mixed**
  - **USA PMAX** is the strongest current paid engine, with **23 conversions** in the last 7 days on about **R7,044.55** spend, or roughly **R306/conversion**.
  - **USA Search** is still effectively dead right now, with **0 impressions, 0 clicks, 0 conversions** in the last 7 days even after launch and the Maximize Clicks switch.

- **Measurement quality:** **off track**
  - GA4 is still structurally contaminated by mixed hosts and polluted conversion definitions.
  - We have a better interpretation now, but not yet a clean measurement system.

- **Lifecycle / automation rebuild:** **mixed**
  - We made the right strategic move by retiring MailerLite and Loops.
  - Sequenzy is the rebuild path, but the live setup is not operating yet.

- **Customer truth / support signal:** **mixed**
  - Gleap is clearly the right operational system to bed into the stack.
  - The skill groundwork is done, but live validation and daily operating use still need wiring.

## What we achieved last week

- Recovered **Google Ads production API access** through the new GCP project and got live reporting/writes working again.
- Refreshed **USA PMAX** assets and launched a new **USA Search** campaign for tighter core-intent traffic.
- Built out the **South Africa Google Ads** test path, then cleaned it up so active South Africa traffic points back to the **Oviond homepage** instead of a throwaway test lander.
- Switched both new Search campaigns from **Maximize Conversions** to **Maximize Clicks** to force initial delivery and start gathering real query data.
- Exposed the **real GA4 problem**: mixed-property noise, polluted conversion metrics, and a suspect Display story that should not be trusted at face value.
- Clarified the **retention thesis**: this is mostly a voluntary churn / perceived-value / trust problem, not just a billing-failure problem.
- Retired **MailerLite and Loops** as live lifecycle platforms and set **Sequenzy** as the clean rebuild path.
- Built first-draft operating skills for **Sequenzy** and **Gleap**, so the stack is getting more usable even before live credentials are fully wired.
- Simplified the **Meta** setup into a survivable interim state while tracking remains unreliable.

## Where we are now

### Revenue and commercial health
- The business still looks like a company with enough market, but not enough consistency yet.
- Revenue pace over the last 7 days softened even while invoice count rose, which suggests we should stay paranoid about quality and retention, not celebrate raw activity.
- Stripe delinquency is not the main fire. The bigger issue is still customer stickiness and saveable churn.

### Trials and conversion pressure
- Trial count is not collapsing, which is good.
- But **10 trials expiring inside 3 days** means the immediate conversion window matters this week.
- The smartest short-term move is not “more traffic at any cost.” It is better activation, rescue, and follow-through on the trials already in the pipe.

### Paid acquisition
- **USA PMAX** is still the best live acquisition signal.
- **South Africa PMAX** produced **7 conversions** in the last 7 days on about **R1,471.04** spend, or roughly **R210/conversion**, but South Africa still needs to be judged on downstream quality, not cheap front-end CPA alone.
- **Display Remarketing USA** is paused and deserves to stay treated like waste-risk. In the last 7 days it burned about **R1,121.88** for **0 conversions**.
- **South Africa Search** barely exists yet, with only **1 click** and no conversions.
- **USA Search** still needs delivery help fast, otherwise it is just dead weight.

### Meta
- Meta is currently weak.
- Current heartbeat read showed **R688.10 spend** for only **2 landing page views**, which is roughly **R344.05 per LPV** and materially above the threshold.
- That is not a scale signal. That is a keep-it-tight signal.

### Measurement and insight quality
- We are in a better strategic position because we now know the old reads were lying to us.
- But until conversion definitions and acquisition segmentation are cleaned up, GA4 should stay in the “useful for pattern-spotting, not truth” bucket.

### Lifecycle and customer signal systems
- We made the right cleanup call by killing the old lifecycle sprawl.
- But a reset is not a rebuild. Sequenzy needs to become live.
- Gleap also needs to move from researched capability to real operating workflow so support, roadmap, help center, and churn language all feed back into growth decisions.
