  - status: enabled
  - `primary_for_goal: true`

  - default value: `2500`
- PMAX campaign `PMAX- Start Trial USA` (`21330916203`):
  - bidding: `MAXIMIZE_CONVERSIONS`
  - goal config level: `CAMPAIGN`
  - biddable goal confirmed as `SUBMIT_LEAD_FORM`
  - budget: `R1000/day`
  - last 7 days: `R4,943.98` spend, `154` clicks, `9` lead-form conversions, approx `R549.33` CPA
  - all 9 lead-form conversions landed between `2026-03-30` and `2026-04-02`
  - device split for trial-signup conversions: `7` desktop, `2` mobile
  - asset-group audit showed many low/unknown headlines and square images, but the campaign is already producing real lead-form conversions
- Display campaign `Display Remarketing USA Feb 2026` (`23607074353`):
  - bidding: `MAXIMIZE_CONVERSIONS`
  - goal config level: `CUSTOMER`
  - ad group has `optimized_targeting_enabled: true`
  - audience lists include trial / demo / all-users remarketing lists, but optimized targeting means it is not behaving like strict remarketing
  - top spend placements include low-quality mobile apps / junk inventory such as TextNow, imo, random app-store placements, weak websites, and YouTube channels
  - last 7 days: `R3,852.57` spend, `2,543` clicks, `0` trial-signup conversions
  - last 14 days: `R8,318.68` spend, `0` trial-signup conversions
  - last 30 days: `R18,685.52` spend, only `18` trial-signup all-conversions total, implying about `R1,038` cost per trial across the whole window
  - last recorded trial-signup conversions from display were on `2026-03-18`; none after that
  - display reporting is heavily polluted by low-value conversion activity, especially `page_view`, so the current conversion column is not trustworthy for spend decisions
- Strategic conclusion from the audit:
  - PMAX is the only currently proven client-acquisition engine in the account
  - the Display campaign is currently a waste-risk and should not keep receiving meaningful budget until rebuilt with strict goals / targeting / exclusions
  - if the true account budget ceiling is `R1000/day`, current combined campaign setup is over-allocated and should be tightened immediately

## Google Ads operating stance after the fresh audit

- Chris wants Google Ads optimized specifically for free-trial signups / `SUBMIT_LEAD_FORM` with a hard practical budget of about `R1000/day` total and very low tolerance for wasted spend.
- Immediate recommendation from the audit:
  - keep PMAX live as the only current proven acquisition campaign
  - pause or near-zero the Display remarketing campaign until rebuilt properly
  - keep PMAX on `MAXIMIZE_CONVERSIONS` for now and do not force a target CPA yet because recent true-conversion volume is still thin
- Last-7-day combined spend across the two live campaigns was about `R8,796.55` total, or roughly `R1,256.65/day`, so the account was effectively overspending versus the intended cap.
- If/when more budget can be deployed beyond PMAX, the next scale path should be high-intent Search around agency reporting / reporting-software alternative terms rather than the current broad Display setup.
- If Display is ever rebuilt, the safe pattern is: campaign-level `SUBMIT_LEAD_FORM` goal only, `optimized_targeting` off, aggressive app/junk placement exclusions, and a very small test budget first.
- Cloud Resource Manager API was later enabled in the fresh project too, but that only improves diagnostics; it does not materially change Google Ads delivery or strategy.
