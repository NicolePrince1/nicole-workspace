# 2026-03-28

## Attribution Hub build started

- Created a new Railway-targeted app at `projects/oviond-attribution-hub/`.
- The app is a Next.js + TypeScript internal tool that treats Oviond backend events as the source of truth, stores them in Postgres, and fans them out to Meta, GA4, Google Ads, and Stripe-aware flows.
- Current MVP includes:
  - canonical event schema and validation
  - Postgres event ledger + delivery-attempt log
  - ingest API with API key auth
  - Meta Conversions API adapter
  - GA4 Measurement Protocol adapter
  - Google Ads delivery scaffold / disabled-safe adapter
  - Stripe webhook ingestion
  - replay endpoint for re-dispatching stored events
  - protected dashboard login using `ADMIN_TOKEN`
  - Railway setup docs in the project README
- `npm run lint` and `npm run build` both passed after implementation.
