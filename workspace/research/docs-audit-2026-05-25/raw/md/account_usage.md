# Usage

> See what your agency has used against your plan's limits.

The **Usage** panel shows how much of your plan's quota you've consumed this billing period. Access it from **Account → Usage**.

## What's tracked

| Metric              | Limit                                                    |
| ------------------- | -------------------------------------------------------- |
| **Clients**         | Number of active clients (archived clients don't count). |
| **Projects**        | Total reports and dashboards across all clients.         |
| **Team members**    | Active users in your account.                            |
| **API requests**    | Calls to the Oviond REST API this month.                 |
| **Emails sent**     | Automation deliveries this month.                        |
| **PDF generations** | Exported PDFs this month.                                |

Each metric shows **used / limit** with a progress bar. If you're close to a limit, Oviond surfaces a warning in the app header.

## Over-limit behavior

* **Hard limits** (clients, projects, team members) block new creation once exceeded — existing records keep working.
* **Soft limits** (API requests, emails, PDFs) temporarily throttle or queue new activity — they don't remove anything.

See [Plans](/billing/plans) for limit details per plan tier, or [Subscription](/billing/subscription) to upgrade.
