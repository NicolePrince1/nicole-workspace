# List Automations

> View, pause, or edit every scheduled delivery.

The **Automations** screen lists every scheduled delivery in your account, grouped by project. Each row shows the automation's name, schedule, next run time, last-run status, and a pause toggle.

## Row actions

| Action              | Result                                                                            |
| ------------------- | --------------------------------------------------------------------------------- |
| **Pause / Unpause** | Toggles whether the automation fires on its next scheduled run. See below.        |
| **Edit**            | Opens the automation for editing (schedule, recipients, subject, body).           |
| **Test**            | Sends the automation immediately — see [Test Automation](/automations/test).      |
| **History**         | Opens the run log — see [Run History](/automations/history).                      |
| **Delete**          | Permanently removes the automation. See [Delete Automation](/automations/delete). |

## Pause / Unpause

Paused automations stay in the list but skip their scheduled runs. Paused is a soft state — toggle it back on any time to resume. Useful when a client is on hold, or you're switching report templates mid-month.

## Filters

* **Project** — show only automations for a single project.
* **Status** — filter to active, paused, or last-run-failed.
