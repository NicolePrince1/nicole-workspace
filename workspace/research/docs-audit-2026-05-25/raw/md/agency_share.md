# Share Project

> Publish an agency report via a shareable public link.

Share an agency report with a teammate, stakeholder, or prospect via a public URL. The recipient opens the link in any browser — **no Oviond account required** — and sees live data.

<Steps>
  <Step title="Open the agency report">
    From the Agency screen.
  </Step>

  <Step title="Click Share">
    The Share button sits in the top-right of the editor. A full link and a short link are both generated.
  </Step>

  <Step title="Enable the public link">
    Toggle **Public link** on.
  </Step>

  <Step title="Copy the link">
    Full URL or short URL — short URLs are easier to share in chat or email.
  </Step>

  <Step title="Optional: password-protect">
    Enable **Password** under the share settings and set a passphrase. Recipients are prompted for it before the report loads.
  </Step>
</Steps>

## Custom domain

If you've set up a [Custom Domain](/custom-domains/list), shared links use your white-label domain instead of `app.oviond.com`. Logo, favicon, and brand colors render on the share page automatically.

## Live vs snapshot

Public links render live data every time they're opened. To send a static snapshot, use [PDF Export](/automations/pdf).

<Warning>
  Anyone with the link can view the report. Use password protection if the agency report contains sensitive internal data.
</Warning>
