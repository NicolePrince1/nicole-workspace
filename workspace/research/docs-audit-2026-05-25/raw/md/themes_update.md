# Update Theme

> Edit an existing theme's colors, fonts, or styling tokens.

Updating a theme changes **every project** that uses it. Project-level data isn't affected — just the visual tokens.

<Steps>
  <Step title="Open the Themes screen">
    From the left sidebar.
  </Step>

  <Step title="Open the theme editor">
    Click the theme, or choose **Edit** from its three-dot menu.
  </Step>

  <Step title="Change tokens">
    Update any of: name, primary color, chart colors, font, border radius, shadow.
  </Step>

  <Step title="Preview">
    The preview panel shows a sample chart and KPI tile using the current settings.
  </Step>

  <Step title="Save">
    The theme is re-applied to every project that uses it on next render.
  </Step>
</Steps>

<Tip>
  If you change the **primary color** without manually overriding the chart palette, Oviond re-derives the chart colors from the new primary. Override individual chart colors before saving if you need them pinned.
</Tip>
