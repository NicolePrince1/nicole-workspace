# Create a Client

> Add a new client workspace in a few fields.

From the Clients list, click **+ Add Client**. Only **Name** is required — the rest can be filled in later.

<Steps>
  <Step title="Name">
    The client's name appears in the sidebar, on report headers, and in the client picker. You can rename it later without affecting existing reports.
  </Step>

  <Step title="Website (optional)">
    When you add a website, Oviond automatically fetches a screenshot and uses it as the client's card thumbnail. You can refresh the screenshot from the client's settings.
  </Step>

  <Step title="Timezone & currency (optional)">
    Sets the default locale for every report under this client — dates, scheduled delivery windows, and money widgets all respect these values.
  </Step>

  <Step title="Manager (optional)">
    Assign a team member as the primary contact. Useful for filtering activity and routing alerts.
  </Step>

  <Step title="Theme & folder (optional)">
    Pick a default [Theme](/themes/list) for new projects, and drop the client into a [Folder](/clients/folders) to organize it with related clients.
  </Step>
</Steps>

Once saved, the client has an empty workspace. Next steps are usually:

1. [Connect a Data Source](/data-sources/connect) for this client
2. [Create your first Project](/projects/list) — a Report or Dashboard
