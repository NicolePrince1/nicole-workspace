# List Templates

> Browse your account's templates and the shared Oviond library.

The **Templates** screen shows every template available to you — those you've created yourself, and the shared library of templates Oviond provides out of the box.

## Sources

| Source             | What it contains                                                                 |
| ------------------ | -------------------------------------------------------------------------------- |
| **My Templates**   | Templates you or your team created from your own projects.                       |
| **Oviond Library** | Shared templates maintained by Oviond — usable by every account without copying. |

## Filters

* **Search** — match by template name.
* **Data sources** — filter to templates that use specific integrations (e.g. templates that include Google Ads widgets).
* **Type** — Reports only or Dashboards only.
* **Sort** — by ranking, name, or creation date.

## What's on a template card

| Element                | Description                                   |
| ---------------------- | --------------------------------------------- |
| **Thumbnail**          | Preview image uploaded with the template.     |
| **Name & description** | Short summary of what the template is for.    |
| **Data-source badges** | Which integrations the widgets inside expect. |

Click a template to preview it before applying to a project — see [Apply Template](/templates/apply).
