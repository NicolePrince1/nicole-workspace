# Folders

> Group related media files into folders.

Media folders keep a growing library navigable. Common patterns: one folder per client (*Acme — Logos*), per type (*Icons*, *Screenshots*), or per report (*Q1 Cover Images*).

<Note>
  Media folders are a separate system from [Client folders](/clients/folders). They only organize items inside the Media library — they don't scope media to any specific client.
</Note>

## Create a folder

<Steps>
  <Step title="Open Media">
    From the left sidebar.
  </Step>

  <Step title="Click + New Folder">
    In the Media sidebar or toolbar.
  </Step>

  <Step title="Name the folder">
    Enter a descriptive name.
  </Step>

  <Step title="Save">
    The folder appears in the Media sidebar, ready to receive items.
  </Step>
</Steps>

## Move items into a folder

From any media tile, click the three-dot menu and choose **Move to folder**, then pick the target. Multi-select also supports bulk move — check multiple tiles, then click **Move** in the action bar.

## Rename a folder

Click the three-dot menu next to a folder name and choose **Rename**.

## Delete a folder

Open the folder's three-dot menu and choose **Delete**.

<Note>
  Deleting a folder **does not delete the media items inside it**. Items return to the unorganized pool and remain accessible from **All Media**.
</Note>
