# Share Project

> Publish a project via a shareable public link.

Share a project with a client, stakeholder, or prospect via a public URL. The recipient opens the link in any browser — **no Oviond account required** — and sees the report with live data.

<Steps>
  <Step title="Open the project">
    From the Reports screen, click the project.
  </Step>

  <Step title="Click Share">
    The Share button sits in the top-right of the editor. The full link and a short link both get generated.
  </Step>

  <Step title="Enable the public link">
    Toggle **Public link** on.
  </Step>

  <Step title="Copy the link">
    Copy the full URL or the short URL. Short URLs are easier to share in chat and email.
  </Step>

  <Step title="Optional: password-protect">
    Enable **Password** under the share settings and set a passphrase. Recipients are prompted for it before the report loads.
  </Step>
</Steps>

## Custom domain

If you've set up a [Custom Domain](/custom-domains/list), shared links use your white-label domain instead of `app.oviond.com`. Your logo, favicon, and brand colors render on the share page out of the box.

## Live vs snapshot

Public links render live data every time they're opened. To send a static snapshot (e.g. end-of-month), use [PDF Export](/automations/pdf) instead.

<Warning>
  Anyone with the link can view the report. Use password protection if the report contains sensitive client data.
</Warning>
