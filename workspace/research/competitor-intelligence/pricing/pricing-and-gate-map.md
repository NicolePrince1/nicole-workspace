# Pricing and Gate Map

Last updated: 2026-05-07

## Executive read

The market makes agencies pay attention to the wrong things. Competitors rarely price only by the agency's real business unit — the client. They commonly gate by source credits, connected accounts, dashboards, reports, users, destinations, API access, AI usage, refresh frequency, data history, support, white label, or custom domains.

Oviond's launch pricing should feel almost boring by contrast:

> Pick your client count. Every core reporting feature is included.

That is commercially strong because agency owners hate discovering later that white label, custom domains, API, extra users, dashboards, reporting automation, or AI summaries sit behind a plan upgrade.

## Gate taxonomy

| Gate type | What it means | Competitors using this pattern | Oviond opportunity |
|---|---|---|---|
| Client count | Pay by number of clients/projects/campaigns | AgencyAnalytics, ReportGarden, Raven, Octoboard-style agency tools | Fine if transparent. Oviond should make this the main/only public buying dimension. |
| Source credits / data accounts | Pay per connected ad account, property, store, profile, or connector account | Whatagraph, Swydo, Porter, Dataslayer, Two Minute Reports, Windsor/Coupler variants | Avoid making agencies count every platform account before buying. |
| Dashboards/reports | Plans limit dashboards, reports, or report templates | DashThis, Klipfolio, some SEO/report tools | Include enough reporting views by client so agencies do not feel punished for serving clients well. |
| Data sources/connectors | Plans limit which connectors or how many source types | Supermetrics, Funnel, Databox, some SEO suites | Include core connectors cleanly; be clear about any premium/coming-soon integrations. |
| Users/seats | Plans charge or limit internal users/client seats | Power BI, Tableau, many enterprise tools; some reporting tools at entry tiers | Unlimited team members stays a strong agency-friendly claim if operationally safe. |
| White label | Branding removal, custom logos, CSS, white-label portals | AgencyAnalytics entry gate, Klipfolio add-ons, Databox add-on/plan, SE Ranking Agency Pack, BrightLocal/Sprout/Hootsuite variants | Make white label feel included, not special. |
| Custom domain | Client/report domains often higher-tier or add-on | AgencyAnalytics higher tiers, Klipfolio add-on, SE Ranking Agency Pack, Whatagraph higher plans | Strong differentiator if bounded self-service works reliably. |
| API | API often higher-tier/enterprise | AgencyAnalytics Agency Pro, Whatagraph Max/Public API, Supermetrics Management API Enterprise, Semrush Business/API, SE Ranking API add-on, Polymer API from high price | API/MCP access included can be a quiet technical wedge; define fair use. |
| MCP/AI | AI summaries, chat, MCP, analyst agents, AI response limits | Databox, Semrush, SE Ranking, Supermetrics, Dataslayer, Coupler, Windsor, Two Minute Reports, NinjaCat, ThoughtSpot | Do not shout AI. Include enough AI/API/MCP capability to prove future-readiness. |
| Refresh/history | Higher refresh frequency or longer history on higher tiers | Supermetrics, Dataslayer, Polymer, TMR, BI tools | Keep default refresh/history adequate for monthly agency reporting; be transparent. |
| Support/onboarding | Priority support, CSM, concierge setup, implementation | AgencyAnalytics Enterprise, Whatagraph Max, Databox add-ons, Funnel/Improvado/TapClicks/NinjaCat sales-led | Offer switcher concierge around launch; reserve true enterprise support for Enterprise. |
| Annual lock-in | Best pricing or required access requires annual billing | Whatagraph paid tiers, Funnel, SE Ranking add-ons, many SaaS tools | Monthly transparency can be a trust signal; annual discounts are fine but avoid coercive framing. |

## AI/MCP threat pricing pattern

AI-native tools rarely replace the whole agency reporting workflow, but they make pieces cheaper and faster:

- **ChatGPT/Claude/Gemini + Sheets/Looker**: cheapest ad hoc analysis and narrative drafting.
- **Julius/Polymer**: upload/connect data, ask questions, generate charts.
- **ThoughtSpot/Power BI/Tableau/Looker AI**: enterprise expectation-setting for conversational/agentic analytics.
- **Google Analytics MCP / Google Ads MCP / Two Minute Reports MCP**: live marketing data inside AI assistants.

Oviond's defense is not to out-hype them. It is to expose secure, scoped Oviond reporting data to AI tools while owning the durable client-reporting loop: connect, template, brand, schedule, review, share, archive, explain.

## Pricing-page implications for Oviond

1. Make the slider the hero: **number of clients**.
2. Explicitly show what is included on every paid plan:
   - white label
   - custom domains with clear fair-use bounds
   - dashboards and reports
   - templates
   - scheduled reporting
   - client portals
   - API access
   - MCP/AI workflow access, framed quietly
   - team members
3. Use “no feature gates” carefully: if usage limits exist, frame them as fair-use/abuse protection, not plan upsells.
4. Add comparison calculators by pricing model, not named attacks: “client-count pricing vs source-credit pricing vs dashboard pricing.”
5. Keep Enterprise for real enterprise needs: SSO, audit logs, advanced support, procurement, custom agreements, scale guarantees.
6. Add a migration/grandfathering note for existing customers before anyone can infer a forced price hike.
