# List Projects

> Browse every account-level agency report.

The **Agency** screen lists every agency report in your account. These aren't scoped to a client, so they don't appear under any client's workspace.

## Filter and search

* **Search** — match by project name.
* **Type filter** — show only **Reports** or only **Dashboards**.
* **Sort** — switch between recent activity, name, and date created.

## What's on a card

| Element         | Description                        |
| --------------- | ---------------------------------- |
| **Name**        | Click to open the editor.          |
| **Type**        | Report or Dashboard badge.         |
| **Last edited** | Timestamp of the most recent save. |

## Quick actions

From any agency-report card's three-dot menu:

* **Open** — enter the editor.
* **Duplicate** — see [Duplicate Project](/agency/duplicate).
* **Share** — see [Share Project](/agency/share).
* **Archive** — see [Archive Project](/agency/archive).
