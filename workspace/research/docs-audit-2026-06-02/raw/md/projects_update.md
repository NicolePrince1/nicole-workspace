# Update Project

> Rename, reassign, and configure project settings.

Project settings cover everything outside the report canvas — name, client assignment, date range defaults, and PDF export options.

<Steps>
  <Step title="Open the project">
    From the Reports screen, click the project you want to edit.
  </Step>

  <Step title="Open project settings">
    Click the **Settings** icon in the editor or choose **Project Settings** from the project's three-dot menu.
  </Step>

  <Step title="Rename">
    Edit the **Project name** field. The new name immediately appears on the shared link and PDF exports.
  </Step>

  <Step title="Reassign the client">
    Use the **Client** dropdown to move the project to a different client.
  </Step>

  <Step title="Set the default date range">
    Pick a fixed range or a relative range (like *Last 30 Days*) — every widget that uses the project-level date inherits this.
  </Step>

  <Step title="Configure PDF export">
    Under **PDF**, set paper size, orientation, and any header / footer content for exports. See [PDF Export](/automations/pdf).
  </Step>

  <Step title="Save">
    Click **Save** to apply.
  </Step>
</Steps>
