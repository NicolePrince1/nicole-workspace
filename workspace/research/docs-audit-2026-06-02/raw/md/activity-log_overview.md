# Activity Log

> Audit every meaningful action taken in your account.

The **Activity Log** records who did what and when, across projects, clients, integrations, users, and billing events. Use it to:

* Trace when a report was created, updated, or deleted
* See which team member invited a user or changed a role
* Correlate a data anomaly with a recent integration reconnect

Entries retain the actor (`user_id`), target (`resource_type`, `resource_id`), and timestamp. Filter by user or resource type from the Activity Log view.
