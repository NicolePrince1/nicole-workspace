# List Themes

> Browse every theme in your account and in the Oviond library.

The **Themes** screen shows every theme you can apply to a project or client.

## Sources

| Source             | What it contains                             |
| ------------------ | -------------------------------------------- |
| **My Themes**      | Themes you created.                          |
| **Oviond Library** | Neutral starter themes maintained by Oviond. |

## What's on a theme card

| Element          | Description                                          |
| ---------------- | ---------------------------------------------------- |
| **Name**         | Shown in pickers and in the project sidebar.         |
| **Color swatch** | Primary color + first few chart colors as a preview. |
| **Font preview** | Sample text in the theme's font.                     |

## Quick actions

From a theme card's three-dot menu:

* **Edit** — see [Update Theme](/themes/update).
* **Apply** — see [Apply Theme](/themes/apply).
* **Delete** — see [Delete Theme](/themes/delete).
