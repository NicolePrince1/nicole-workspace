# Profile Settings

> Update your own name, avatar, email, and preferences.

Profile settings are personal to your login — every user edits their own, regardless of role.

## What you can change

| Setting           | Notes                                                                                                            |
| ----------------- | ---------------------------------------------------------------------------------------------------------------- |
| **Full name**     | Shown throughout the app and in activity-log entries.                                                            |
| **Avatar**        | Upload any square image. Displayed in the sidebar and on shared pages when you're the account's primary contact. |
| **Email address** | Changing your email re-verifies it — Oviond sends a confirmation link to the new address.                        |
| **Language**      | UI language — see the [supported languages](#supported-languages) below.                                         |
| **Theme**         | Light / Dark / System. Doesn't affect how reports render for clients.                                            |

## How to access

Click your avatar or initials in the bottom-left of the sidebar, then **Profile**.

## Supported languages

Oviond's UI ships in 14 languages. Pick the one that matches your team's preference from the **Language** dropdown — the entire app (navigation, buttons, forms, tooltips) re-renders immediately.

| Code    | Language                 |
| ------- | ------------------------ |
| `en`    | English (default)        |
| `af-ZA` | Afrikaans (South Africa) |
| `bg`    | Bulgarian                |
| `da`    | Danish                   |
| `de`    | German                   |
| `el`    | Greek                    |
| `es`    | Spanish                  |
| `fr`    | French                   |
| `it`    | Italian                  |
| `nl`    | Dutch                    |
| `pl`    | Polish                   |
| `pt`    | Portuguese               |
| `ro`    | Romanian                 |
| `tr`    | Turkish                  |

<Note>
  The language setting is **per-user** — each team member can pick their own, and what you choose doesn't affect how client-facing reports render. Reports always display in the language they were authored in.
</Note>

<Note>
  Your role, allowed projects, and account-level settings are managed by an Admin or Owner — not from your profile. See [Update User](/users/update).
</Note>
