# Pages

> Organize a project into tabs — one page per section of the report.

A **page** is a tab inside a project. Each page has its own grid layout and holds any number of widgets. Viewers switch between pages to explore different sections — e.g. one page per marketing channel.

Every project starts with at least one page. Add more at any time.

## Add a page

<Steps>
  <Step title="Open the project">
    Click the project on the Reports screen.
  </Step>

  <Step title="Click + Add Page">
    The button sits in the page tab bar at the top of the editor.
  </Step>

  <Step title="Rename it">
    Double-click the tab to rename. Use clear names like *Google Ads — Overview* or *SEO Performance*.
  </Step>
</Steps>

## Reorder and hide pages

* **Reorder** — drag a page tab left or right to change its position.
* **Hide** — from the page's three-dot menu, toggle **Hidden** to drop it from the client-facing view without deleting it.

## Delete a page

Open the page's three-dot menu and choose **Delete**. Every widget on the page is removed with the page.

<Tip>
  Plan pages before adding widgets. Grouping related data per page (e.g. one page per channel) keeps reports scannable for clients.
</Tip>
