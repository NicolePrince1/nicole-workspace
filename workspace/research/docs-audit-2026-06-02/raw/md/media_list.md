# List Media

> Browse every file in your Media library.

The Media library lists every uploaded file — images, logos, icons, and documents — in a grid view, most-recent-first. Click any item to preview it, copy its URL, or move it into a folder.

## Views

* **All Media** — the unfiltered grid.
* **By Folder** — narrow to a single [folder](/media/folders) from the sidebar.
* **Search** — match by file name.
* **Sort** — switch between recent activity, name, and file size.

## What's on a tile

| Element          | Description                                                     |
| ---------------- | --------------------------------------------------------------- |
| **Thumbnail**    | Rendered preview of the file.                                   |
| **Name**         | The file's display name — editable via [Rename](/media/update). |
| **Size**         | File size in KB/MB.                                             |
| **Folder badge** | The folder this item is in, if any.                             |

## Quick actions

From any tile's three-dot menu:

* **Preview** — open the full file in a lightbox.
* **Copy URL** — copy the direct file URL (useful for referencing in widgets or external tools).
* **Move to folder** — see [Folders](/media/folders).
* **Rename** — see [Rename Media](/media/update).
* **Delete** — see [Delete Media](/media/delete).
