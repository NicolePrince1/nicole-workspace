# List Custom Data

> Every CSV dataset uploaded to your account.

The **Custom Data** list shows every dataset you've uploaded — name, row count, column count, file size, and upload date.

## Filters

* **Search** — match by dataset name.
* **Sort** — by name, upload date, or row count.

## What's on a row

| Column       | Description                                   |
| ------------ | --------------------------------------------- |
| **Name**     | The label you gave the dataset on upload.     |
| **Columns**  | Number of columns parsed from the CSV header. |
| **Rows**     | Number of data rows (excluding the header).   |
| **Size**     | Raw file size of the CSV.                     |
| **Uploaded** | Timestamp of the most recent upload.          |

## Quick actions

From any row's three-dot menu:

* **Preview** — see the first 50 rows.
* **Update** — see [Update Custom Data](/custom-data/update).
* **Duplicate** — see [Duplicate Custom Data](/custom-data/duplicate).
* **Delete** — see [Delete Custom Data](/custom-data/delete).
