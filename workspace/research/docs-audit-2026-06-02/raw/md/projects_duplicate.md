# Duplicate Project

> Copy a project — pages, widgets, and settings — under a new name.

Duplicating creates an exact clone of a project, including every page, widget, and setting. Useful for:

* Reusing a reporting structure for a different client or month
* Creating a variant without editing the original
* Testing layout changes on a copy before applying to the live report

<Steps>
  <Step title="Find the project">
    On the Reports screen, locate the project you want to copy.
  </Step>

  <Step title="Open the project menu">
    Click the three-dot menu next to the project.
  </Step>

  <Step title="Choose Duplicate">
    Oviond creates a copy named `Copy of [original]`.
  </Step>

  <Step title="Rename and reassign">
    Open the copy's settings to give it a new name and, if needed, assign it to a different client.
  </Step>
</Steps>

<Tip>
  If you want a reusable starting point rather than a one-off copy, [create a Template](/templates/add) from the project instead — templates keep a live library you can re-apply to any future project.
</Tip>
