# Duplicate Project

> Copy an agency report — pages, widgets, and settings — under a new name.

Duplicating creates an exact clone of an agency report, including every page, widget, and setting. Useful for:

* Spinning up a new variant (e.g. quarterly vs. monthly version)
* Testing layout changes on a copy before applying to the live report
* Creating a forked version for a different internal team

<Steps>
  <Step title="Find the report">
    On the Agency screen, locate the agency report you want to copy.
  </Step>

  <Step title="Open the menu">
    Click the three-dot menu next to the report.
  </Step>

  <Step title="Choose Duplicate">
    Oviond creates a copy named `Copy of [original]`.
  </Step>

  <Step title="Rename">
    Open the copy's settings to give it a new name. See [Update Project](/agency/update).
  </Step>
</Steps>

<Tip>
  If you want a reusable starting point rather than a one-off copy, [create a Template](/templates/add) from the agency report instead — templates keep a library you can re-apply to any future project.
</Tip>
