# Apply Theme

> Use a theme on a project or set it as a client's default.

Themes apply at two levels:

| Level           | What it affects                                                        |
| --------------- | ---------------------------------------------------------------------- |
| **Per-project** | Applies the theme only to the selected project.                        |
| **Per-client**  | Sets the theme as the default for every new project under that client. |

## Apply to a project

<Steps>
  <Step title="Open the project">
    From the Reports screen.
  </Step>

  <Step title="Open Project Settings">
    From the project's three-dot menu or the settings icon.
  </Step>

  <Step title="Pick a theme">
    Under **Theme**, select from My Themes or the Oviond Library.
  </Step>

  <Step title="Save">
    Every chart, KPI tile, and background re-renders using the theme's tokens.
  </Step>
</Steps>

## Set a client default

<Steps>
  <Step title="Open the client's workspace">
    From the Clients list.
  </Step>

  <Step title="Open Client Settings">
    Click the three-dot menu on the client header.
  </Step>

  <Step title="Pick a default theme">
    Under **Default Theme**, select one. New projects under this client start with this theme.
  </Step>
</Steps>

<Note>
  Changing a client's default theme does **not** change existing projects — only new ones. Update existing projects individually if you want them retroactively rethemed.
</Note>
