# List Clients

> View and filter every client in your workspace.

The **Clients** list is your home dashboard — every client you've created appears here, most-recently-active first. Each client card shows the client's name, website screenshot, project count, and folder (if assigned).

## Views

* **All Clients** — the unfiltered list.
* **By Folder** — narrow the list to a single folder from the sidebar.
* **Search** — type in the search bar to match by client name.
* **Sort** — switch between recent activity, name (A–Z), and date created.

## What's on a client card

| Element           | Description                                                                                                  |
| ----------------- | ------------------------------------------------------------------------------------------------------------ |
| **Name**          | Shown in the card header and the sidebar.                                                                    |
| **Screenshot**    | Auto-generated from the client's website. Click **Refresh Screenshot** from the client's menu to regenerate. |
| **Project count** | Number of active reports and dashboards in the client's workspace.                                           |
| **Folder badge**  | The folder the client is assigned to, if any.                                                                |

<Note>
  The Clients list respects your **allowed projects** if your role is limited. Team members with restricted access only see clients whose projects they're authorized to view.
</Note>

## Quick actions

From any client card's three-dot menu:

* **Open** — enter the client's workspace.
* **Rename** — edit the client's display name inline.
* **Move to folder** — see [Group Clients](/clients/folders).
* **Archive** — soft-delete the client; see [Archive Client](/clients/archive).
