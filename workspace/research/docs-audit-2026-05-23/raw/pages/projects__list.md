# List Projects

> Browse, search, and filter every project in your workspace.

The **Reports** screen lists every project across every client. Use it to find, open, or manage any report or dashboard.

## Filter and search

* **Search** — type in the search bar to match by project name.
* **Client filter** — narrow to a single client using the **Client** dropdown.
* **Type filter** — show only **Reports** or only **Dashboards**.
* **Sort** — switch between recent activity, name, and date created.

## What's on a project card

| Element         | Description                                |
| --------------- | ------------------------------------------ |
| **Name**        | Click the name to open the project editor. |
| **Client**      | Shows which client the project belongs to. |
| **Type**        | Report or Dashboard badge.                 |
| **Last edited** | Timestamp of the most recent save.         |

<Tip>
  Use consistent project naming — including month and year (e.g. *Acme — Paid Social — March 2026*) — to make the list searchable as it grows.
</Tip>
