# Apply Template

> Use a template as the starting point for a new project.

Applying a template copies its pages, widgets, and settings into a brand-new project. You then wire the widgets to the client's specific data sources.

<Steps>
  <Step title="Start a new project">
    From the Reports screen, click **+ New Project**. See [Add Project](/projects/add).
  </Step>

  <Step title="Choose Use a Template">
    In the create dialog, pick **Start from template** instead of **Blank**.
  </Step>

  <Step title="Pick a template">
    Browse **My Templates** or the **Oviond Library**. Preview any template to see the page and widget layout before applying.
  </Step>

  <Step title="Assign a client & create">
    Assign the project to a client and click **Create**. Oviond copies every page and widget into the new project.
  </Step>

  <Step title="Wire data sources">
    Open each widget and pick the matching data source for your client — the template ships with the widget structure but no live connection.
  </Step>
</Steps>

<Tip>
  If a template expects a data source you haven't connected for this client, Oviond flags the widget with a **Connect required** badge. See [Connect Data Source](/data-sources/connect).
</Tip>
